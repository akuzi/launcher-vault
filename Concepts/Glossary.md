# Glossary

Core launcher terminology.

- **Home app / launcher** — the app that draws the home screen. See [[What is a Launcher]].
- **App drawer** — the full list of installed apps, usually opened by swiping up. Some launchers omit it (drawer-less).
- **Dock / hotseat** — the persistent row of icons at the bottom, shown across all home pages.
- **Workspace / home pages** — the swipeable pages holding icons and widgets.
- **Widget** — an interactive/glanceable mini-app embedded on the home screen (clock, weather, calendar). See [[Widgets]].
- **At a Glance** — Google's home-screen widget surfacing contextual info (next event, weather, travel). Native to [[Pixel Launcher]]; cloned by [[Lawnchair]].
- **Icon pack** — a downloadable set of replacement icons that a launcher applies app-wide. See [[Icon Packs and Theming]].
- **Adaptive icons** — Android's icon format (since Oreo) with a separate foreground/background, enabling uniform masks (circle, squircle, teardrop).
- **Themed / monochrome icons** — Material You tinted icons matching wallpaper colors (Android 13+).
- **Material You / Monet** — Android's dynamic color system that derives a palette from the wallpaper.
- **Gesture** — a swipe/tap/pinch on the home screen mapped to an action (open app, expand notifications). See [[Gestures as Input]].
- **Scrub bar / fast scroll** — an alphabet index for jumping in a long app list (e.g. [[Niagara Launcher]]).
- **Folder** — a container grouping multiple app icons.
- **Smartspace** — Google's term for the At a Glance / contextual card area.
- **Drawer-less** — a launcher with no separate app drawer; all apps live on home pages or in a list.
- **Profile / mode (launcher)** — a switchable saved home-screen configuration (layout, apps, theme) the launcher swaps between by context. See [[Profiles]].
- **Work Profile** — an Android-managed, isolated space for work apps/data; shown as a badged Work tab in the drawer. See [[Profiles]].
- **Private Space** — Android 15+ hidden, lockable space for sensitive apps. See [[Profiles]].
- **Backup/restore** — exporting a layout config to migrate or recover. A key switching-cost factor.
- **Notification dots / badges** — unread indicators on icons.
- **Quickbar / search bar** — persistent search entry.
- **Prime / Pro** — the common name for a launcher's paid unlock tier.
