# AIO Launcher

"All-In-One." An **information-dense** launcher: the home screen is a stack of text widgets (system stats, feeds, finance, notifications, tasks) rather than icons.

## At a glance
- **Category**: [[Launcher Taxonomy|Info-dense]]
- **Retrieval model**: search + text widgets (see [[Launcher Mental Models]])
- **Developer**: Evgeny Zobnin

## Pricing
- **Free** core; **Premium** via one-time purchase and/or subscription for advanced widgets/customization. See [[Pricing Comparison]].
- Model: mixed [[Monetization Models|one-time / subscription]].

## Key features
- Home screen built from **compact text blocks**: monitoring (RAM/CPU/battery), notifications, calendar, RSS/news, finance/crypto, web search, scripts.
- Highly **scriptable / extensible** (custom widgets, plugins).
- Search line for apps and actions.
- Low visual chrome, dense layout.

## Mental model fit
The **opposite** of [[Digital Wellbeing and Minimalism]] — maximal **push / ambient information** (axis 5, [[Launcher Mental Models]]). Everything glanceable at once; optimised for the user who wants a dashboard, not a calm screen.

## Best for
"I want my phone to be a control panel." Compare with [[Microsoft Launcher]] (feed-style info, more consumer) and the polar opposite [[Olauncher]].
