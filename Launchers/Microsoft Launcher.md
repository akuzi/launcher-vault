# Microsoft Launcher

A **free, full-featured** launcher whose strategic purpose is to bind Android users into the **Microsoft 365 / Windows** ecosystem.

## At a glance
- **Category**: [[Launcher Taxonomy|Ecosystem-tied]]
- **Retrieval model**: spatial grid + feed (see [[Launcher Mental Models]])
- **Developer**: Microsoft

## Pricing
- **Free**. Model: [[Monetization Models|vendor-funded / strategic]] — earns indirectly by driving Microsoft service usage and Windows–Android continuity.

## Key features
- **"Your Feed"** — glanceable panel with calendar, To-Do, recent docs, news, sticky notes.
- **Continuity with Windows** — Phone Link integration, "continue on PC".
- Microsoft 365 / To-Do / Outlook / OneDrive hooks.
- Reasonable customization: grid, icon packs, gestures, themes.
- Bing-backed search.

## Mental model fit
Strong on the **push / ambient information** axis ([[Launcher Mental Models]]) via the feed — opposite end from minimalist launchers. Its value is *integration*, not customization depth.

## Trade-offs
- The feed and Microsoft/Bing integration mean it profiles activity ([[Privacy and Launchers]]); "free" here is paid in data + ecosystem lock-in.
- Less appealing if you don't live in Microsoft's tools.

## Best for
"I use Microsoft 365 / Windows and want my phone to plug into it." Compare with [[Pixel Launcher]] (Google's equivalent strategic play).
